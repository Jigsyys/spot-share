FriendsModal — Redesign avec Groupes

Les groupes sont actuellement introuvables et leurs invitations n'ont aucune UI. 3 propositions pour corriger ça.

🐛 Problèmes actuels avec les Groupes

— Création de groupe : uniquement depuis le dropdown de la carte (invisible)

— Gestion membres : icône ⚙️ dans ce même dropdown, 3 clics minimum

— Invitations de groupe : comptées dans le badge Amis mais aucune UI pour les accepter/refuser

— Pas de lien entre Groupes et Sorties

Groupes = onglet dédié. Sorties et classement rejoignent le feed Activité. Simple et équilibré.
