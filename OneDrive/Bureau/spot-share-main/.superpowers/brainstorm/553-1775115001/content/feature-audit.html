Section 2 — Audit des features existantes

Chaque feature notée sur Impact utilisateur · Clarté UX · Problèmes détectés

➕ Ajouter un spot

★ Killer feature
URL Instagram/TikTokIA Google PlacesAjout manuelVisibilitéGroupesExpiration

✅ UX excellente, l'ajout par URL est différenciant. L'IA qui pré-remplit tout est le gros point fort du produit.

⚠️ Manque : aucun retour chiffré après ajout ("tu as ajouté 3 spots ce mois — tu es #2 au classement !"). Opportunité de gamification ratée.

🗺️ Carte + Filtres

Solide
Amis / Groupes / Mes spotsFiltre ami+catégorieMode 3DDark/LightClustering

✅ Filtres bien pensés. Le mode groupe et filtre par catégorie sont puissants.

🔴 Le filtre ami+catégorie (icône SlidersHorizontal) n'est visible qu'en mode Amis actif. Beaucoup d'utilisateurs ne le découvriront jamais.

⚠️ Manque : aucun badge "Nouveau" visible sur les pins des spots récents (< 48h). En phase B, l'utilisateur ne sait pas ce qui a changé depuis sa dernière visite.

📌 Panel Spot (clic sur un pin)

Bon contenu
Carousel photosLikes (❤️)VisitesHorairesNavigationPartage

✅ Riche en info. Le long-press sur ❤️ pour voir les likers est une belle petite feature.

🔴 "J'ai visité ce lieu" est sous le fold — la plupart des users ne scrolleront pas jusqu'à ce bouton.

⚠️ Manque : pas de commentaires/tips. Likes + visites c'est bien mais un commentaire court ("ask pour le menu du soir!") crée de la valeur pour les autres.

⚠️ Manque : pas de "spots similaires" ou "autres spots de cet ami" en bas du panel.

🔍 Explorer

Sous-exploité
Recherche texteFiltre catégorieTri distanceClassement mensuelMode Surprise 🎲

⚠️ Feature puissante mais l'icône 🔍 + label "Explorer" n'est pas self-explaining pour un nouvel utilisateur.

🔴 Le mode "Surprise" (spot aléatoire) est une feature fun pour l'engagement mais quasi-invisible dans l'UI.

🔴 En phase A (groupe fermé), Explorer retourne les mêmes spots que la carte. Peu de valeur ajoutée perçue.

⚠️ Manque : "Top spots cette semaine" / trending. Donner une raison de revenir chaque semaine.

👥 FriendsModal — onglets Amis / Classement / Invitations

Trop dense
Liste amis en ligneProposer sortiePodium mensuelDemandes amisSorties invitées

⚠️ Les 3 onglets contiennent des features de natures très différentes regroupées de façon non intuitive.

🔴 "Invitations" est un mauvais nom : cet onglet contient demandes d'amis + sorties reçues + mes sorties + demandes envoyées. → Renommer en "Activité".

🔴 Le classement mensuel est la feature d'engagement la plus forte — l'enterrer dans FriendsModal > onglet 2 est une erreur. Peu de gens le trouvent.

⚠️ Manque : pas de feed d'activité globale ("X a ajouté un spot hier", "Y a liké ton spot").

🗓️ Sorties (Outings)

Feature invisible
Créer une sortieInviter des amisChat par invitationAccepter / Décliner

🔴 Fonctionnalité très originale et différenciante, mais complètement introuvable pour un nouvel utilisateur.

🔴 Flow de création : FriendsModal → onglet "Amis" → trouver un ami dans la liste → "Proposer une sortie". 4 clics minimum, sans aucun hint dans la nav.

🔴 Pas d'entrée directe depuis le panel d'un spot. Pourtant "Proposer d'aller à cet endroit ?" depuis un spot serait le flow le plus naturel.

⚠️ Manque : raccourci direct dans la nav ou depuis le panel spot. C'est la feature qui peut le plus créer de la fidélisation.

🏘️ Groupes

Bonne idée, flow flou
Créer un groupeFiltrer la carteSpots privés par groupe

⚠️ Le concept est bon (groupe de famille, collègues, etc.) mais l'invitation de membres n'est pas évidente.

⚠️ Manque : lien entre groupes et sorties. Créer une sortie depuis un groupe serait naturel.

👤 Profil

Fonctionnel
Mes spotsMes likesFollowers / FollowingGhost modeDark mode

✅ Bien structuré. Ghost mode apprécié pour la vie privée.

⚠️ Manque : stats engageantes ("X personnes ont visité tes spots", "Tu es #2 ce mois"). Le profil devrait être une source de fierté et de motivation.

⚠️ Manque : partage de profil (lien public). Pourtant le partage de spot existe déjà — même pattern, plus de surface de croissance.

Résumé des priorités :

🔴 Critique : Rendre les Sorties trouvables · Feed d'activité · Bouton visite visible

🟡 Important : Classement plus visible · Explorer plus engageant · Stats profil

🟢 Amélioration : Commentaires spots · Partage profil · Badges "Nouveau"
